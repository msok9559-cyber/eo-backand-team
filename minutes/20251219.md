# 작업 목표
- 메인페이지 강사님 피드팩받은거 전부 수정
- 회원정보 수정페이지 완료 및 제출
- 비밀번호 페이지 전체 완료
- 질문페이지 수정 할꺼 할수있으면 작업하기
## 작업 회고
- 질문페이지 사이드바 열었을때 본문 영역을 밀리게 설정하는게 좀많이 힘들었다
- 처음 작업 시작할때 기초대로 작업해야 많은걸 바꾸지않고 수정이 가능하다는걸 느낌
- 먼가 수정 될수록 다른게 오류나고 오류나서 고치는게 점점 힘들어짐
- 생각했던것보다 틀이 어느정도 잡혀서 괜찮다.
## 내일 목표
- 질문페이지 내정보 페이지 만드는거 비밀번호 찾기 페이지의 색감을 바꾸기
- 전제척인 페이지를 메인페이지에 맞춰서 색감 통일
- 작업완료후 모든작업물 연결
- 리드미 작성 시작
- 재웅씨가 시간이 남으면 창규씨 작업 도움
- 최대한 코드 확인후 수정
- css 틀 무너지거나할시 다시 수정
- 회원정보 수정페이지 작업완료 버튼 작동시 다른페이지로 이동
