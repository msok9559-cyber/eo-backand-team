## 오전회의 내용
- 2차 목표 질문페이지 보안 작업 착수
- 비밀번호 찾기 페이지 작업 착수
- 회원가입 페이지 틀은 잡았으나 문제가 많아 보안 필요
- 회원가입 페이지 css 틀에 문제 발생으로 인한 보안 필요

## 작업 목표

## 작업 회고

- 비밀번호 찾기페이지 에서 은근 자바스크립트로 구현해야될 부분이많다.
- 팀원들간에 js코드 공유로 배워가는 점이 많아 좋음
- 코드에 대한 이헤력이 좀더 숙달되는 느낌이다.
