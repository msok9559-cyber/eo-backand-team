## 오전회의 내용
- 마이페이지 작업 시작
- 질문페이지 보안내용 작업
- 회원가입 페이지 작업 시작

## 작업 목표

- 질물내역페이지 채팅창을 전체화면으로 수정 및 이미지 수정
## 작업 회고

- 리셋 css 적용후 틀이 무너지는 현상으로 인하여 css 구조를 바꿈
- 회원 정보 수정페이지 미숙함으로 인한 팀원들과 함께 코드 수정
- 멘토님 피드백 덕분에 미흡한 부분이 보여 보안 작업이 필요하다는걸 느낌

- 회원 수정페이지 작업 착수 (미완료)
