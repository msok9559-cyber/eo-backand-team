# 1차프로젝트 2팀회의록
- 날짜: 2025년12월 16일
- 팀명 :  JDJS = jipdanjisung(집단지성)
- 참여자: 박민성(팀장),김재웅,정창규
## 오전회의 내용
- 오늘도 스트레스 받지않고 목표를 정한대로 열심히 나아가기
- 주말 푹 쉬었으니 이번주도 힘내서 작업하기
## 작업 목표
- 질문페이지 색감 메인페이지하고 동일하게 변경
- 마이페이지 모든 작업 완료
- 리드미 작성
## 작업 회고
- 쿠키값을 공부하기 위해 열심 찾아보느라 오늘 하루를 날림
- 회원정보 수정페이지 css수정부분이 복잡함
- 질문페이지 이미지 찾고 호버효과를 주고 채팅장 수정을 중 원하는 값이 나오지않아서 내일도 좀 손봐야될듯함
## 내일 목표
- 쿠키값을 받아오고 받아온 쿠키값을 이용하여 질문페이지 로그인시에만 질문페이지 사용하게끔 변경
- 로그인페이지 js 작업 시작
- 회원정보 수정페이지 완성
- 입력창 폰트사이즈 조절 및 화면 작아질때 입력창이 사이드바위로 올라오는 문제 해결
## JDJS팀 규칙
- 팀원간 싸움 금지
- 안된다 외치기 금지
- 포기란 없다. (지피지기면 백전백승)
- 항상 웃으면서 작업하기!
