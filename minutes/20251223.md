# 1차프로젝트 2팀회의록
- 날짜: 2025년12월 16일
- 팀명 :  JDJS = jipdanjisung(집단지성)
- 참여자: 박민성(팀장),김재웅,정창규
## 오전회의 내용
- 쿠키값 공부하기
- 로컬 저장소로 변환 하여 로그인값 저장
## 작업 목표
- 로그인페이지 js코드 완성하기
- 질문하기페이지 마지막 수정하기
- 코드 오류 수정 및 마이페이지 완성
- 로그인 js와 쿠키값을 활용하여 질문페이지 막기
## 작업 회고
- 쿠키값을 사용할려고 햇으나 서버가 없으면 쿠키값이저장되지않아 사용이 불가 판정
- 로컬값을 이용하여 사용하였음
- 회원가입 페이지 비밀번호 찾기 페이지 등에 들아가는 js 가 많다.
- 수정 페이지 에서 회원페이지 합칠려고 했으나 생각외로 작업이 안됬다.
## 내일 목표
- 마이페이지 최종본 확인 후 수정
- 파일 이름 명칭 통일 및 그로인한 파일 연결 다시하기
- 리드미파일 최종 정리 및 링크 사이트 전부 연결
- 회원가입 페이지 자바스크립트 완료
- ppt 작업 준비 
## JDJS팀 규칙
- 팀원간 싸움 금지
- 안된다 외치기 금지
- 포기란 없다. (지피지기면 백전백승)
- 항상 웃으면서 작업하기!
