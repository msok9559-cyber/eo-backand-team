## 🎯 프로젝트 목표
이스트소프트 백엔드 과정의 팀 프로젝트로  
**AI 기반 포털 사이트 화면 설계**를 목표로 합니다.

### 📌 구현 페이지
- 메인 페이지 🔍  
- 질문 결과 페이지 💬  
- 회원 로그인 페이지 🔐  
- 회원 가입 페이지 📝  
- 마이페이지 (정보 확인/수정) 👤  
- 비밀번호 찾기 페이지 🔑  
- 질문 내역 확인 페이지 📄  

---

## 👍 프로젝트 소개
본 프로젝트는 실제 AI 포털 서비스를 벤치마킹하여  
**HTML / CSS / JavaScript**를 활용한  
프론트엔드 화면 구현에 집중하였습니다.

팀원 간 역할 분담과 협업을 통해  
팀프로젝트 진행 경험을 목표로 합니다.
